## Installation

## Installation

```bash
git clone https://github.com/eszczepan/Cookie-clicker
npm install
npm run start
```

## Available scripts

| Command         | Description            |
| --------------- | ---------------------- |
| `npm run start` | Open local server      |
| `npm run build` | Create optimized build |
| `npm run test`  | Run tests              |
| `npm run eject` | Eject CRA              |
